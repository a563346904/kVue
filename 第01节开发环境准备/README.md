#### <a id="vue-cli安装及使用"></a>
#### 安装
```javascript
npm i -g @vue/cli
```
#### 使用

#### 快速原型开发
##### 首先安装全局扩展
```javascript
npm install -g @vue/cli-service-global
```
##### 启动开发服务器
```javascript
vue serve ./Hello.vue
```
##### 创建项目
```javascript
vue create project-name
```
#### 使用插件
##### 安装vue-router
```javascript
vue add router
```
